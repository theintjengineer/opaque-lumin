# Lumin

A glassmorphism-inspired theme for Zed.

Lumin preserves the visual language of Vesper Blur: translucent surfaces, blur effects, dark panels, and minimal UI — while replacing the editor syntax highlighting with a Cursor-inspired palette.

## Features

* Blur-based UI surfaces
* Transparent editor appearance
* Dark minimal interface

## Build From Source

You can install Lumin locally without waiting for extension registry updates.

### 1. Fork the repository

Fork this repository

### 2. Clone your fork

```bash
git clone https://github.com/frypan05/Lumin.git
```

### 3. Install as a development extension

Open **Zed**.

Navigate to:

```text
Extensions
→ Install Dev Extension
```

Then select the cloned **Lumin** folder.

Zed will load the theme immediately.

### 4. Enable the theme

Open:

```text
Settings → Theme
```

Select:

```text
Lumin
```
